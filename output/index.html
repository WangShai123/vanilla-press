组件示例 文档站首页

文档站首页

这是基于 markdown-it 的静态文档流程示例。构建命令会读取 input/**/*.md，按目录结构输出到 output/**/*.html，并生成一份共享 CSS 与一份共享 JS。

Tabs

执行构建命令：

npm run build

构建后可以直接打开 output/index.html 查看页面。

默认输入目录是 input，默认输出目录是 output。也可以直接传参：

node src/build.js input output

Accordion

组件语法先被解析为静态占位 DOM，最终页面加载后再由 vanilla-jui 实例化。

CSS 合并为 styles.css，运行时代码打包为 assets/doc-runtime.js。

Offcanvas

侧栏内容

这里是由 Markdown 写出的 offcanvas 内容，页面运行时会调用 createOffcanvas() 创建实例。

import { createOffcanvas } from 'vanilla-jui';

const panel = createOffcanvas({
direction: 'right',
content: '<p>更多内容</p>',
});

panel.show();
